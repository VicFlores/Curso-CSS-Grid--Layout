Curso de ReactJS

React hace que sea sencillo crear interfaces de usuario interactivas. Diseñe vistas simples para cada estado en su aplicación, y React actualizará y renderizará de manera eficiente los componentes correctos cuando cambien sus datos. Las vistas declarativas hacen que su código sea más predecible y más fácil de depurar.
